# Pasta de Dados

Por favor, faça o download do dataset do Spotify no Kaggle:
- **Dataset:** [Spotify Tracks Dataset](https://www.kaggle.com/datasets/maharshipandya/spotify-tracks-dataset)
- **Nome do arquivo:** Salve o arquivo CSV baixado como `dataset.csv` nesta pasta (`data/dataset.csv`).

O dataset original possui cerca de 114 mil faixas, 125 gêneros e 20 atributos de áudio e metadados.
